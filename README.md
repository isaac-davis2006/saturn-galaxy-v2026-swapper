## What Saturn Galaxy Does

Saturn Galaxy is a Windows-based desktop utility focused on Fortnite cosmetic swapping. From one interface, it lets users manage appearance items including skins, emotes, backblings, pickaxes, and wraps. It also includes legacy OG Fortnite asset support, which makes it useful for those who want access to older cosmetic collections in addition to newer content.

A major part of the project is keeping pace with Fortnite releases by updating memory offsets after patches. In addition to that update workflow, the tool offers AI-driven cosmetic suggestions and a multilingual, responsive UI, giving Windows users a flexible way to browse and handle cosmetic selections.
